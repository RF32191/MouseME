# MouseMe

Turn your iPhone or iPad into a wireless trackpad, keyboard, media remote, and TV controller for your Mac — over your local Wi‑Fi.

## Platforms

- **iPhone / iPad** — controller (trackpad, keyboard, TV remote, connect)
- **Mac** — receiver (listens for phone connections, moves cursor via Accessibility)

## Requirements

- iPhone or iPad and Mac on the same Wi‑Fi network
- macOS Accessibility permission for the Mac receiver
- iOS Local Network permission for discovery
